# Налаштування публікації (українською)

Це коротка інструкція як створити самодостатній виконуваний файл для презентації.

1) Опублікувати для Windows x64 в режимі Release (single-file):

```powershell
dotnet publish HEK.sln -c Release -r win-x64 /p:PublishSingleFile=true /p:PublishTrimmed=false --self-contained true
```

2) Після цього результат буде в підпапці `UniversalLogAnalyzer\bin\Release\net8.0-windows\win-x64\publish\` або в `publish/` кореня проекту, залежно від конфігурації скриптів.

3) Якщо потрібно зменшити розмір, можна включити тримінг (`/p:PublishTrimmed=true`), але це може вимагати додаткових перевірок під час демонстрації.

4) Альтернативно, використайте `PublishPortable.ps1` (якщо є), щоб зібрати і підготувати артефакти одним скриптом.

Порада: перед презентацією відтестуйте exe на машині з чистим середовищем, щоб перевірити, що UI та експорти працюють коректно.
