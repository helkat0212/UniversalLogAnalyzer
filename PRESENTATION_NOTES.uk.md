# Нотатки для презентації (українською)

Коротка структура презентації (5–7 хвилин demo):

1) Мета програми
- Швидкий огляд: парсинг мережевих конфігів та логів для виявлення проблем і створення Excel-репорту.

2) UI — коротке live demo
- Відкрити приклад файлу (Huawei/Cisco)
- Показати Raw Data вкладку
- Показати Results (Interfaces, BGP, VLANs)
- Показати виявлені Anomalies

3) Excel Export
- Згенерувати і відкрити Excel (показати, що аркуші зʼявляються тільки коли є дані)

4) Технічні деталі (1 слайд)
- .NET 8 WPF, парсерна архітектура `ILogParser` (vendor-specific parsers), уніфікована модель `UniversalLogData`.

5) Що ще заплановано
- Покращення парсерів (VRRP, eth-trunk, vlan batch parsing, краща VPN/VRF ідентифікація)
- Додати юніт-тести для нових випадків парсингу

6) Q&A / коротке тестування на live sample

Поради під час demo:
- Мати під рукою 2–3 файли різних вендорів з помітними аномаліями
- Якщо Excel відкривається повільно, заздалегідь згенеруйте файл і покладіть у `publish/` для швидкого відкриття.
